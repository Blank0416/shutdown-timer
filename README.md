# Shutdown Timer ⏰

极简 Windows 定时关机工具 — 浏览器操作，精准到秒。

## 使用

```bash
node server.js
```

浏览器打开 `http://127.0.0.1:4399`

## 功能

- **倒计时关机** — 输入分钟数，自动关机
- **定时关机** — 选具体时间（次日如果已过）
- **快捷按钮** — 15m / 30m / 1h / 2h / 立即关机
- **实时倒计时** — 调用 Windows `shutdown` 命令，真关机
- **取消关机** — 随时中止
- 零依赖 — 只需 Node.js

## 原理

`shutdown /s /t <秒>` → 后台倒计时 → 到达后 Windows 关机

## 技术栈

- Node.js 原生 `http` 模块
- SSE 实时推送到浏览器
- 纯前端，无框架
